# GROMACS Molecular Dynamics Simulation Agent

A Python CLI chatbot that uses **RAG (Retrieval-Augmented Generation)** to answer GROMACS molecular dynamics questions accurately. Before each Gemini API call, relevant documentation is retrieved from a local vector database, ensuring the agent produces real commands — never hallucinated ones.

The knowledge base is scraped directly from the **official GROMACS documentation** during ingestion, so the agent always has up-to-date, accurate information.

## Features

- 🔍 **RAG-powered** — retrieves relevant docs before every response
- ⚡ **Gemini Flash** — fast, free-tier generation with streaming output
- 📚 **11 official doc pages** scraped and indexed automatically
- 💾 **Smart caching** — scrape cache (7-day) + embedding cache (hash-based)
- 🎨 **Rich terminal UI** — spinners, progress bars, ASCII banner
- 🛡️ **Robust error handling** — retries, circuit breaker, graceful degradation

## Prerequisites

- Python 3.9+
- pip
- A free Gemini API key from [aistudio.google.com](https://aistudio.google.com)
- Internet access (first run only, for scraping docs)

## Quick Start

```bash
git clone https://github.com/username/gromacs-agent
cd gromacs-agent
pip install -r requirements.txt
python setup.py        # first-run wizard: API key + validation
python ingest.py       # scrape docs + build vector DB (run once)
python agent.py        # start the chatbot
```

> **Note:** `ingest.py` requires internet access on first run to scrape GROMACS documentation. Subsequent runs use the local cache (valid for 7 days). The `docs/` directory is auto-generated by `ingest.py` and is **not** committed to the repo.

│                             ▼                                   │

┌─────────────────────────────────────────────────────────────────┐

│                                                                 │

│                                       │                         │
│                                       ▼                         │

```

## Commands

| Command | Description |
|---------|-------------|
| `/help` | Show available commands |
| `/clear` | Clear conversation history |
| `/save` | Save last response to file |
| `/sources` | Show sources from last query |
| `/reindex` | Re-run ingest pipeline |
| `/structures` | Rescan input_structures/ |
| `/key` | Update Gemini API key |
| `/exit` | Exit the agent |

## Error Handling

- **429 Rate limit**: Retry 3× with exponential backoff (5s, 10s, 20s)
- **401 Invalid key**: Error message, return to prompt
- **500/503 Server error**: Retry once, then return to prompt
- **Circuit breaker**: 5 consecutive errors → clean exit with instructions
- **No ChromaDB**: Friendly message + exit
- **No internet**: Falls back to cached docs if available
